# odin-recipes


## Introduction
This project is part of the Foundations course at The Odin Project. It is a simple recipe website that will demonstrate my ability to structure a basic website using HTML and manage a project using Git.

## Skills Demonstrated
By the end of this project, I will have practiced:
- Creating and linking HTML files.
- Using Git for version control (commits, pushing to GitHub).
- Structuring content with headings, lists, and images.
